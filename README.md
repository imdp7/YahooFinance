- Authentication - AWS Cognito (AWS pool)
- UI Library - AWS Design (Cloudscape v3)
- Backend - Node.js MongoDB
- Redux Store
- UseCollection Hooks
- Payments and billing through Stripe.js API - Validation of cards and banks
